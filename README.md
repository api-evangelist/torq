# Torq

Torq is an enterprise-grade AI-driven hyper-automation security platform that enables security operations teams to triage, investigate, and respond to threats at machine speed. The platform supports no-code, low-code, and full-code security automation with 300+ integrations, AI HyperAgents, autonomous case management, and regional deployment options in the United States and European Union.

## API

The Torq REST API provides programmatic management of workspace resources including workflows, users, secrets, triggers, and webhooks. All API keys are scoped to a specific workspace.

- **US Endpoint**: https://api.torq.io
- **EU Endpoint**: https://api.eu.torq.io
- **Rate Limit**: 50 requests per second per API key
- **Timestamp Format**: RFC 3339 (yyyy-MM-ddTHH:mm:ss.SSSZ)
